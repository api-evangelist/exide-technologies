# Exide Technologies (exide-technologies)
Exide Technologies is a global producer and recycler of lead-acid batteries used in transportation and industrial applications, serving automotive, motive power, network power, and other markets. Headquartered in Milton, Georgia, the company operates manufacturing and recycling facilities worldwide.
